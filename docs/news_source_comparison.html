1 · Cost trajectory (monthly $ vs daily active users)

B (Google) wins at small/medium scale — at ~8k DAU it crosses below C. Above ~100k DAU, C catches up because it's flat-priced until the enterprise tier; B keeps growing with Claude usage. D is always free, but it's an event database — not a headline feed — so it complements rather than replaces.
